# Sound Swap

This community based web app was designed for musicians who would like to rent or lease musical instruments. Users would be free to rent or lease an instrument for as little as an hour or a day at a time. 


## Getting started

This project runs with a Ruby on Rails api download here: 
https://github.com/pkayres/sound-swap-api

In the project directory, you can run:

### `npm start`
Open [http://localhost:3001](http://localhost:3001) to view it in the browser.

### This site was built with: 
- [React](https://reactjs.org) The JavaScript library used
- [Redux](https://redux.js.org) - Global State Management
- [THUNK](https://github.com/reduxjs/redux-thunk) - Middleware for Redux
